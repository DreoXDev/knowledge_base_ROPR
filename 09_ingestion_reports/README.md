# Ingestion Reports - ROPR

Questa cartella raccoglie tutti i report di analisi degli asset e dei batch.

## Struttura

- `batch_reports/`: Report aggregati per batch di studio.
- `asset_reports/`: Report dedicati all'ingestione e all'analisi del singolo file.
- `final_audit.md`: Audit finale sulla qualità e completezza della KB prima dell'esame.
