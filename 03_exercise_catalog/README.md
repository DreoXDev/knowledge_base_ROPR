# Exercise Catalog - ROPR

Questo catalogo raccoglie e categorizza tutti gli esercizi d'esame e le relative varianti.

## Struttura

- `EXERCISE_INDEX.md`: Indice generale degli esercizi.
- `EXERCISE_TYPES.md`: Tassonomia delle tipologie di esercizio.
- `by_topic/`: Esercizi suddivisi per argomento.
- `by_source/`: Esercizi suddivisi per fonte (es. appello specifico).
